Temple Recognition Agent control files
======================================

Recommended:
  Double-click agent_control.bat

Menu:
  [1] Start Agent
  [2] Stop Agent
  [3] Open Web Page
  [4] Refresh Status
  [0] Exit

Quick actions:
  start_agent.bat  - start the Agent
  stop_agent.bat   - stop the Agent
  run_agent.bat    - old direct start script

Agent URL:
  http://127.0.0.1:7860

If stop says "access is denied", right-click agent_control.bat and choose
"Run as administrator", then select Stop Agent again.
